# 项目需求文档：多交易所资产余额 H5 看板

## 1. 项目目标

开发一个私人使用的 H5 / PWA 工具，用来查看多个加密货币交易所账户余额。

用户可以登录后台，在用户中心添加交易所 API Key，只允许读取余额，不支持交易、提现、划转等操作。

网站部署在 Cloudflare Pages，后端使用 Cloudflare Pages Functions / Workers，数据库使用 Cloudflare D1。

项目需要支持添加到 iPhone 桌面，像 App 一样打开。

---

## 2. 核心原则

1. 只做余额查看，不做交易。
2. API Key 只允许 Read Only 权限。
3. API Secret 绝不能出现在前端。
4. 所有交易所 API 请求必须由后端完成。
5. 前端只调用自己的后端 API。
6. 用户必须登录后才能查看资产。
7. API Key / Secret 必须加密后存入数据库。
8. 页面风格简洁，适合手机浏览。

---

## 3. 技术栈要求

### 前端

- React
- Vite
- TypeScript
- Tailwind CSS
- PWA 支持
- 部署到 Cloudflare Pages

### 后端

- Cloudflare Pages Functions 或 Workers
- TypeScript
- Hono 或原生 Worker Router
- D1 数据库
- Web Crypto API 做加密
- JWT 登录鉴权

### 数据库

Cloudflare D1，SQLite 语法。
